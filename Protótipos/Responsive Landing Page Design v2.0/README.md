# Karton Landing Page v2.0

Esta é a landing page responsiva do projeto **Karton**, desenvolvida com **Next.js 15**, **Tailwind CSS v4** e **Framer Motion**. 

O design foi originalmente baseado no protótipo do Figma: [Responsive Landing Page Design](https://www.figma.com/design/fFJTfcYIQiTRiFUcdcXHPN/Responsive-Landing-Page-Design).

## 🚀 Tecnologias

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Como rodar o projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado. Recomendamos o uso do **pnpm** para gerenciamento de pacotes.

1. Instale as dependências:
   ```bash
   pnpm install
   # ou
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   pnpm run dev
   # ou
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 📦 Estrutura do Projeto

- `src/app`: Contém as rotas e o layout principal (App Router).
- `src/components`: Componentes reutilizáveis, incluindo as caixas 3D animadas.
- `src/lib`: Utilitários e funções auxiliares.
- `public`: Arquivos estáticos (imagens, ícones, etc).
